Welcome to this repository of **Machine Learning and AI examples and projects implemented in Python**!  

This repository is organized into two main folders:  

- **`examples/`** – Self-contained notebooks demonstrating key ML/AI concepts.  
- **`projects/`** – Hands-on projects combining multiple techniques and real datasets.  





---

## 📖 Examples

1. **Linear Regression Titanic Survivors**  
   Predict survival chances using **linear regression** on the Titanic dataset.

2. **Classification Flowers**  
   Classify flower species based on petal and sepal measurements.

3. **Hidden Markov Models – Weather Model**  
   Learn **Hidden Markov Models (HMMs)** to model sequences, applied to weather prediction.

4. **Classify Images of Clothing**  
   Use a **neural network** to classify Fashion MNIST clothing images.

5. **CNN Classify CIFAR Images**  
   Build and train a **Convolutional Neural Network (CNN)** for CIFAR-10 image classification.

6. **CNN Cats vs Dogs with MobileNetV2**  
   Apply **transfer learning** with MobileNetV2 to classify cat and dog images.

7. **Text Classification with an RNN**  
   Classify text data using a **Recurrent Neural Network (RNN)**, e.g., sentiment analysis.

8. **Text Generation with an RNN**  
   Train an **RNN** to generate text character-by-character based on a dataset.

9. **Building the Q Table**  
   Implement **Q-learning** for reinforcement learning in a discrete environment.

## 🚀 Projects


### 1. **Rock Paper Scissors**  
A simple game with an **AI agent**, demonstrating reinforcement learning techniques.

### 2. **Cat and Dog Image Classifier**  
Classify images of cats and dogs using **CNNs and transfer learning**. Includes preprocessing, training, and evaluation.

### 3. **Book Recommendation Engine using KNN**  
A **content-based recommendation system** using **k-Nearest Neighbors (KNN)** to suggest books based on user preferences.

### 4. **Linear Regression Health Costs Calculator**  
Predict healthcare costs using **linear regression** on patient data (age, BMI, smoking, etc.).

### 5. **Neural Network SMS Text Classifier**  
Classify SMS messages as spam or ham using a **neural network**, with text preprocessing and vectorization.
